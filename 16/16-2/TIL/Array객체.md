## 자바스크립트의 내장 객체

### Array 객체

- 배열을 다루는 내장 객체
- 초기값이 없는 상태에서 단순히 객체의 인스턴스만 만든다면 new를 사용해 변수에 할당하기
- 배열의 크기를 지정하지 않을수도 있고, 몇 개의 요소가 있는지 크기 지정 가능
- 초기값이 있다면 인스턴스 선언과 요솟값을 한번에 할당 가능
- 배열요소는 프로그램 안에서 얼마든지 추가하거나 삭제 가능→ 요소의 개수를 알고 사용하는 것이 좋음→ Array안에 있는 length 프로퍼티를 사용하면 된다.

```c
let numbers = new Array(); //배열의 크기를 지정X
let numbers = new Array(4); // 배열의 크키를 지정
```

```c
let numbers = ["one", "two", "three", "four"]; //배열 선언
let numbers = Array("one", "two", "three", "four"); // Array객체를 사용해 배열 선언
```

### Array 객체의 메서드

| 종류     | 설명                                                                          |
| -------- | ----------------------------------------------------------------------------- |
| concat   | 기존 배열에 요소를 추가해 새로운 배열을 만듭니다                              |
| every    | 배열의 모든 요소가 주어진 함수에 대해 참이면 true, 아니면 false 반환          |
| filter   | 배열 요소 중에서 주어진 필터링 함수에 대해 true요소만 골라 새로운 배열을 만듬 |
| forEach  | 배열의 모든 요소에 대해 주어진 함수를 실행                                    |
| indexOf  | 주어진 값과 일치하는 값이 있는 배열 요소의 첫 인덱스를 찾음                   |
| join     | 배열 요소를 문자열로 합침, 이때 구분자를 지정 가능                            |
| push     | 배열의 맨 끝에 새로운 요소를 추가한 후 새로운 length를 반환                   |
| unshift  | 배열의 시작 부분에 새로운 요소를 추가                                         |
| pop      | 배열의 마지막 요소를 꺼내 그 값을 결과로 반환                                 |
| shift    | 배열에서 첫번쨰 요소를 꺼내 그 값을 결과로 반환                               |
| splice   | 배열에 새로운 요소를 추가하거나 삭제                                          |
| slice    | 배열에서 특정한 부분만 자름                                                   |
| reverse  | 배열의 배치 순서를 역순으로 바꿈                                              |
| sort     | 배열 요소를 지정한 조건에 따라 정렬                                           |
| toString | 배열에서 지정한 부분을 문자열로 반환, 이때 각 요소는 쉼표로 구분              |

**배열끼리 합치는 concat()메서드**

- 서로 다른 두 배열 2개를 합쳐서 새로운 배열을 만듬
- 어느 배열을 먼저 쓰는가에 따라 기준이 달라짐, 결과 배열의 순서도 달라짐

```jsx
let name = ["세은", "수아", "다연", "지민", "지연", "조을", "소희", "소림"];
let num = [1, 2, 3, 4, 5, 6, 7, 8];

let numFirst = num.concat(name); //1, 2, 3, 4, 5, 6, 7, 8, 세은, 수아, 다연, .....
let nameFirst = name.concat(num); //세은, 수아, 다연, 지민, 지연, ..., 1, 2, 3, ...
```

**배열 안의 요소끼리 합치는 join()메서드**

- 배열 요소를 연결하여 하나의 문자열로 만들어줌
- 각 요소에 원하는 구분자(’/’)를 넣을 수 있음→ 따로 지정하지 않으면 쉽표로 구분

```jsx
const a = ["바람", "물", "불"];
a.join(); // '바람,물,불'
a.join(", "); // '바람, 물, 불'
a.join(" + "); // '바람 + 물 + 불'
a.join(""); // '바람물불'
```

**새로운 요소를 추가하는 push(), unshift()**

- 배열에 새로운 요소를 추가할 때 사용
- 배열 맨 끝에 추가하려면 push(), 배열의 맨 앞에 요소를 추가하려면 unshift() 사용
- ※ 배열 맨 앞과 맨 뒤에 요소를 추가하면 원래 있던 배열이 바뀜
- 호출한 배열의 새로운 [`length`](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/length) 속성

**배열에서 요소를 꺼내는 pop(), shift()**

- 배열에서 뒤에 있는 요소를 꺼낼 때는 pop(), 앞에 있는 요소를 꺼낼 때는 shift()사용
- 배열에서 마지막 요소를 제거하고 그 요소를 반환

**원하는 위치에 요소를 추가&삭제하는 splice()**

- 배열 중간 부분에서 요소를 추가&삭제
- 한꺼번에 요소 2개 이상 추가&삭제
- 괄호 안에 들어 있는 인수에 따라 배열 요소를 삭제하거나 새로운 요소를 추가
- 삭제한 요소로 이루어진 새로운 배열이 결과값으로 반환
- 제거한 요소를 담은 배열을 반환
- 원본 배열 유지X

1. 인수가 1개인 경우

- 괄호 안의 인수는 배열의 인덱스값, 즉 배열의 위치를 가리킴
- 인수가 지정한 인덱스 요소부터 맨 끝까지 삭제

1. 인수가 2개인 경우

- 첫번째 인수는 인덱스값이고, 두번째 인수는 삭제할 요소의 개수

1. 인수가 3개 이상인 경우

- 첫번쨰 인수는 배열에서 삭제할 위치를, 두번쨰 인수는 삭제할 개수, 세번째 인수부터는 삭제한 위치에 새로 추가할 요소 지정

**기존 배열을 바꾸지 않으면서 요소를 꺼내는 slice()**

- pop(), shift()와 달리 시작과 끝 인덱스를 지정해서 요소를 여러개 꺼냄
- 실행 결과 기존 배열이 바뀌지 않음

1. 인수 1개인 경우

- 그 인수를 시작 인덱스로 간주하고, 지정한 요소부터 마지막 요소까지 꺼내 반환

1. 인수가 2개인 경우

- 여러개의 요소 꺼내기 가능
- 첫번째 인수는 배열의 시작 인덱스, 두번쨰 인수는 끝 인덱스의 바로 직전 인덱스 가리캄(슬라이싱과 비슷)
